### This project is a simple frontend application built using Next.js. It provides an easy-to-use interface where users can enter details of a used vehicle and instantly receive a predicted price from the backend API.

## Features:

#### - Clean, minimal UI for entering vehicle attributes.  
#### - Sends the input data to the prediction API.  
#### - Displays the predicted price instantly on the same page.  

## Tools/Libraries/Frameworks Used

### Frontend Framework:

#### - Next.js  
#### - React

### Styling:

#### - Tailwind CSS

### Language:

#### - TypeScript

## How to Use

#### - Open the web app.  
#### - Fill out the form with vehicle details (year, odometer, manufacturer, etc.).  
#### - Click the **Predict** button.  
#### - The predicted price returned by the API will appear on the screen.
